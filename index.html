QuaDramA Viewer

This is an early version of a viewer for dramatic texts. It is based on automatic processing of TEI encoded dramatic texts from the TextGrid repository. You can browse 687 texts. The web application has been developed in the context of the QuaDramA project and will likely not be continued.

→ Go to the text overview ←

Details

The web application offerns six different views of each texts

Text View

The dramatic text itself. Currently, stage directions are not displayed in this view. The view shows only text spoken by a figure. Using the sidebar, you can jump to acts/scenes and/or highlight figures.

Presence View

An interactive version of a configuration matrix. Every utterance is represented in the chart. It allows a quick glance at co-presence of figures and even displays the spoken text when hovering.

Presence View2

An other view on the co-presence of figures. It shows relative portions of spoken words by scene / act.

Figure Statistics

Some simple, descriptive metrics on each figure

Semantic Fields

Using manually constructed dictionaries, we assign a topical score to each figure. The score represents the amount of spoken tokens from a particular topic.

Copresence Network

Interactive co-presence network. Node size is based on amount of spoken tokens per figure. Width of connection represents the (normalized) number of co-present scenes the two figures have.
